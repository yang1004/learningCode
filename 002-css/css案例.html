针对美方的言论，俄外交部发言人扎哈罗娃3月30日回应称，美国试图通过对俄委两国的合法合作实施制裁来恐吓莫斯科，这看起来很荒谬。俄方已明确表明本国专家的赴委目的。美国关于俄专家在委进行某些“军事行动”的猜测毫无根据。扎哈罗娃同时呼吁美国停止“扼杀”委经济和推动该国陷入内战的行为，称这是在公然违反国际法。而对于美俄总统有可能就委内瑞拉局势沟通的问题，俄总统新闻秘书佩斯科夫表示，克里姆林宫并未收到有关提议。

据秘鲁《共和报》3月30日报道，同俄方一样，委内瑞拉防长洛佩斯也表示，任何人都没有必要对俄军队来到加勒比国家感到惊恐。“委俄的军事技术合作自2001年就已开始，我们只是合作”。洛佩斯还就博尔顿的警告回应说，美方称俄军机抵达加拉加斯为“丑闻”，但美国军机此前曾抵达哥伦比亚边境城市库库塔，美国空军在委内瑞拉周边的无线电电子侦察活动大幅增加，“却没人对这种威胁委主权的事情发声”。就俄国防产品出口公司表示，俄方参与建设的直升机飞行员训练中心已于3月29日在委投入使用一事，洛佩斯表示，有了这家训练中心，委内瑞拉今后可以向其他拥有俄罗斯直升机的国家提供飞行训练。这是平等战略伙伴之间、即俄罗斯和委内瑞拉合作的一部分。

面对俄美因委内瑞拉问题而日益紧张的关系，俄议员莫罗佐夫认为，美国不会决定与俄军发生直接冲突，也无法阻止俄委合作。美国众议院外交事务委员会主席恩格尔也称，众议院民主党人不会支持美国军事干预委内瑞拉的选项。但共和党人认为，军事干预是必要的，他们甚至将俄罗斯对委现政府的支持比作莫斯科2015年对叙利亚内战的干预。“这是我们的后院，我们必须保护这里”。
